# Cronus Server Migration

On Wednesday, March 18th beginning at 9:00 AM, SSCS will conduct a migration of the Cronus server to a new compute environment offering more memory, faster processors, and additional resources. Maintenance is expected to conclude before 5:00 PM.

<!-- more -->

The new environment will provide improved performance over the current Cronus cluster. 
You are receiving this notice because you have an authorized account on Cronus.

**What is affected:**

- Cronus will be unavailable from 9:00 AM until maintenance is complete.

**What is NOT affected:**

- `ssdfiles.uchicago.edu` will not be impacted by this change. If you access Cronus 
files on your own computer via ssdfiles, nothing will change.

We will send a follow-up notification once maintenance is complete. If you have any 
questions, please reach out to us at 
[ssc-server-support@lists.uchicago.edu](mailto:ssc-server-support@lists.uchicago.edu).

— Social Sciences Computing Services
