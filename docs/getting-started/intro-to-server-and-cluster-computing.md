# Types of Computing

## Standalone - Laptop or Desktop Computers
- Standalone, 1-user systems are "End User Devices" -- ***EUDs***
- PCs and Macs are familiar to nearly everyone
- All Programs run locally, on the computer in your control
- No network connection required
- Programs run only while the computer is running
- Vulnerable to loss, theft, failure, and budget
