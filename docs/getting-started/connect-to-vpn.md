# Connect to the University VPN (CVPN)

Before you can access the cluster, you must connect to the University of Chicago VPN (CVPN).

1. Install the VPN from [https://vpn.uchicago.edu](https://vpn.uchicago.edu)
2. Use your CNet credentials to configure and connect
3. Ensure the VPN connection is active before proceeding to SSH or file transfer with SFTP, Globus, etc.

!!! note "On-Campus Users"
    If you are accessing the cluster from on-campus, you do not need to connect to the CVPN. VPN is only required when accessing the cluster from off-campus or outside the UChicago network.
